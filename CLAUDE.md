Project: Layman News App

Goal:
Build an AI-powered news app that simplifies articles.

Core Constraints:
- Headlines: 48–52 chars
- Article cards: 28–35 words
- Exactly 3 cards
- Chat responses: 1–2 sentences

Tech:
- React Native (Expo)
- Supabase
- News API
- AI API

Architecture:
- Feature-based structure
- Zustand state management
- Services layer for APIs

Important:
- UI must match mockup
- AI output must follow strict format
